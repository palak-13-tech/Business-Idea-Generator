# Business Idea Generator

## Overview
Business Idea Generator is a Python-based web application that generates business ideas based on user-selected industries and AI.

## Features
- Generate unique business ideas
- AI-powered suggestions
- Simple and user-friendly interface

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
